WHAT IS LORIS?

“LORIS is a modular and extensible web-based data management system that integrates all aspects of a multi-center study: from heterogeneous data acquisition (imaging, clinical, behavior, genetics) to storage, processing and ultimately dissemination.”

What’s involved?

Longitudinal Acquisition, Storage and Curation, Multi-site Data storage, Project management Web-based, open-source Validation, Quality Control, Protocol Checking, Upgrades, Maintenance, Bug Fixes, User Interface, Javascript, Bootstrap, Behavioural, Clinical, Imaging, Genetic Data Multi-Modal Linking, Querying ...No big deal!
